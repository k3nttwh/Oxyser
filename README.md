# Oxyser
Oxyser is a public archive with cloud/mobile/network/web-app/OT/SCADA/IoT/ penetration testing and red team operations notes (In developement)
## Instalation
First time, unzip the oxyser.zip with following command(if u are using linux):
```bash
unzip oxyser.zip
```
## Usage
Second time, open the folder, open the cherry tree file(Oxyser.ctb), or if u want PDF version, open the Oxyser.pdf file
More than that, we have a online version hosted at gitbook, always [here](https://oxysernotes.gitbook.io/oxysernotes-docs)
